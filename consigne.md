# Consigne

## Objectif principal

L’objectif est de moderniser le site tout en respectant les meilleures pratiques en matière de design web et de développement.  
Votre solution devra être performante, esthétiquement agréable, et facile à utiliser pour tous les visiteurs.  

Vous devrez réaliser une mise à jour complète sur le site de **Molengeek** en respectant les consignes et fonctionnalités suivantes :  

---

## Carrousel interactif

- Intégrer un carrousel moderne et attractif pour mettre en avant les contenus importants.  
- Assurez-vous que le carrousel soit fluide et optimisé pour une navigation agréable sur tous les appareils (ordinateurs, tablettes et mobiles).  

---

## Menu burger dynamique

- Ajouter un menu burger intuitif et fonctionnel pour améliorer l’expérience utilisateur sur les appareils mobiles.  
- Le design du menu doit être clair et facile à utiliser, tout en s’intégrant harmonieusement au style général du site.  

---

## Design responsive à 100 %

- Le site doit être entièrement responsive, s’adaptant automatiquement à toutes les tailles d’écran.  
- Veillez à ce que l’interface reste cohérente et ergonomique, quel que soit le support utilisé.  

---

## Contraintes techniques

- **L’utilisation de Bootstrap est strictement interdite.**  
- Tout le code devra être conçu manuellement pour garantir une personnalisation totale.  

---

## Outils requis

### Figma
- Toutes les maquettes doivent être conçues et validées via Figma avant le développement.  

### Notion
- Le suivi du projet devra être organisé sur Notion pour garantir une bonne communication et une gestion efficace des tâches.  

### GitHub
- L’ensemble du code devra être versionné sur GitHub.  
- Vous devrez créer un dépôt dédié au projet, avec des commits clairs et réguliers documentant chaque étape de votre progression.
